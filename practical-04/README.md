## Description
Update project car showroom.

路 Before Loading a home page call get API and take the car's        response and show them in view.

路 On submit add-car dialog in a stand of showing alert now we have to call post API of car data and send all data in it.

路 If the response is 200 then call get API again else show an error message in the alert.

路 Same for the delete button on click call delete API with respective id and if you found 200 then call get API again and update the view. On alert show error.
